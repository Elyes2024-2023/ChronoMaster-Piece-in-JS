1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/chronomaster-piece.git
   cd chronomaster-piece
   ```

   ```bash
   npm install
   ```

3. Create a `.env` file:
   ```
   VITE_WEATHER_API_KEY=your_api_key
   VITE_SOCKET_URL=http://localhost:3001
   ```

   ```bash
   node server/index.js
   ```
